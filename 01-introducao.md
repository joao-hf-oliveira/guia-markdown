## O que é Markdown?
Markdown é uma linguagem de marcação simples.  
Ela serve para formatar textos usando apenas caracteres comuns.

Você escreve o texto de forma simples  
e o Markdown transforma isso em conteúdo organizado.

## Onde o Markdown é usado
- README de projetos no GitHub
- Documentações Técnicas
- Anotações de estudo
- Blogs
- Apostilas
- Sites

## Por que usar Markdown?
- É Fácil de aprender
- Deixa o conteúdo organizado
- Funciona em qualquer editor
- É muito usado por programadores

## Como visualizar 
No VsCode:  
- Ctrl + Shift + V → preview
- Ctrl + Shift + V novamente → volta ao código
