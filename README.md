# Estudo de Markdown

Este repositório foi criado para um estudo independente de Markdown,  
com foco em aprender na prática, organizar conteúdo e boas práticas

O material está dividido em capítulos numerados, abordando desde os  
conceitos básicos até a estruturação de uma apostila utilizando apenas Markdown.

## Objetivo

- Aprender Markdown do zero
- Criar anotações organizadas e legíveis
- Simular uma apostila de estudo
- Usar Markdown de forma profissional

## Conteúdo

O estudo inclui:
- Formatação de texto
- Listas e tarefas
- Links e imagens
- Blocos de código
- Linhas e divisores
- Comentários
- Boas práticas

## Como usar

Abra os arquivos `.md` no VS Code e utilize o preview  
(`Ctrl + Shift + V`) para visualizar o conteúdo formatado  
ou visualize diretamente a formatação no GitHub.

A leitura é sequencial, seguindo a numeração dos arquivos.  
Siga os links de "próximo / anterior" para melhor experiência de usuário.

## Observação

Este material é voltado para estudo pessoal e aprendizado contínuo.
