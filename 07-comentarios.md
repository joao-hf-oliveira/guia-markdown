# Comentários no Markdown

## Objetivo

Aprender a usar comentários  
para fazer anotações que não aparecem no preview.

## Comentário simples
```html
<!-- Isso é um comentário -->

Esse texto aparece no preview.
<!-- O comentário não aparece, só o texto normal -->
```

## Comentário em bloco

```html
<!-- 
Esse comentário pode ter
várias linhas
e serve para anotações
-->
```

## Para que usar comentários?
- Lembretes pessoais
- Explicações pessoais
- Conteúdo para inserir futuramente
- Orgnanização de estudo/desenvolvimento

## O que não é comentário

`//` Isso **não** é comentário.  
`#` Isso **não** é comentário.

## Resumo

- Markdown usa comentário HTML
- Comentários não aparecem no preview
- São ótimos para estudo
